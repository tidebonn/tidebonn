Overføringen er fullført

Tidebønn lever nå fullt og helt på Areopagos sine egne kontoer. Domenet tidebonn.no er kjøpt — siste steg er å peke det mot Vercel. Det gjør du selv via Domeneshop på et par minutter.

Status: Klar — venter på DNS

Handling kreves — DNS hos Domeneshop

Sett opp tidebonn.no

Domenet er allerede lagt til i Vercel fra min side. For at tidebonn.no faktisk skal peke på appen, må DNS-en endres hos Domeneshop. Logg inn på domene.shop og finn tidebonn.no.

1. Slå av «WWW-videresending»

Under «Aktive tjenester» — slå av WWW-videresending. Vercel håndterer videresending fra www → tidebonn.no automatisk.

2. Endre DNS-records

Klikk «Endre DNS». Slett den eksisterende A-recorden som peker til 185.134.245.113 (Domeneshops parkering), og legg inn disse i stedet:

TypeNavn / SubdomeneVerdi / Peker til
A@ (eller blank)216.198.79.1
A@ (eller blank)64.29.17.1
CNAMEwwwcname.vercel-dns.com.

La navnetjenerne (ns1.hyp.net, ns2.hyp.net, ns3.hyp.net) stå som de er — vi trenger ikke endre dem.

3. Vent på propagering

DNS oppdaterer seg vanligvis innen 5-60 minutter. Vercel oppdager automatisk når records er på plass, og utsteder gratis SSL-sertifikat fra Let's Encrypt (~1 min). Etter det er tidebonn.no live.

Daglig drift

Hva som skjer fra nå av

Bjørnulf gjør kodeendringer og pusher til GitHub. GitHub Actions deployer automatisk til Vercel innen ~60 sek. Ingen manuell drift kreves fra dere.

Som eiere kan dere når som helst:

  • Se brukerstatistikk under Admin → Statistikk i appen
  • Administrere brukere under Admin → Brukere
  • Redigere bønner, serier og innholdssider
  • Promotere/degradere andre admins
Eier vs admin: Owner-rollen kan kun settes via Supabase SQL Editor — ikke fra appen. Det er for å verne om eier-rollen mot utilsiktet endring. Admins gis via Admin → Brukere.

Trenger dere hjelp?

Send meg dette

Hvis dere vil at Bjørnulf hjelper med oppsettet, send følgende informasjon:

  1. Hvilket domene dere vil bruke (f.eks. tidebonn.no eller tidebonn.areopagos.no)
  2. Hvor domenet er registrert (Domeneshop, Norid, Cloudflare, GoDaddy, e.l.)
  3. Skjermbilde av Vercel sin DNS-instruksjon (Settings → Domains etter at dere har lagt til domenet)
  4. Skjermbilde av DNS-administrasjons-siden hos registraren (gjerne med eksisterende records skjult/redigert hvis nødvendig)

Med dette kan Bjørnulf gi konkrete instruksjoner uten å trenge innlogging på Areopagos sine kontoer.

Hjelp via skjermdeling

For en del DNS-administrasjons-paneler (særlig Cloudflare) er det like raskt å sette opp via en kort skjermdeling. Avtal et tidspunkt om dere foretrekker det.
